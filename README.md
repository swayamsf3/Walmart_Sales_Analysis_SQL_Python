## Walmart Sales Analysis | SQL & Python

This project focuses on analyzing Walmart retail sales data using SQL and Python.
The objective is to understand sales trends, top-performing product categories,
and region-wise performance to support business reporting.

### Dataset
- Walmart sales dataset containing transaction-level sales information.

### Tools Used
- SQL (MySQL, PostgreSQL)
- Python (Pandas, NumPy)
- Jupyter Notebook

### Project Workflow
- Cleaned and prepared raw sales data using Python.
- Loaded cleaned data into SQL databases.
- Executed SQL queries to analyze sales trends and performance.
- Documented insights for business reporting.

### Key Learnings
- Writing business-focused SQL queries
- Performing data cleaning and basic data exploration using Python
- Analyzing retail sales data for decision-making
